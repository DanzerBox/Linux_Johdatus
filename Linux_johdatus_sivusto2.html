Tehtävä 3.1 Komentorivin käyttö

Tehtävä suoritettu, 21.04.2021
Asensin VirtualBoxin.

VirtualBox oli minulla jo asenettuna, joten tästä kuvasta näkee vain kuinka se asennetaan Windowsille. VirtualBoxin asennus on suht helppoa ja ohjeita löytää helposti.

Asensin Linux Mint 20.

Valitsin uusimman Linux Mint 20 Ulyana, joka on 64-bit Cinnamon versio koko (7,34GB).

Avasin Linux Mint 20.

Sitten kun saa avattua VirtualBoxin, avataan vaan "NEW" kohdasta ja valitaan asetuksista Linuxin ja Ubuntu 64-bit, sillä Mint on Ubuntu pohjainen. Myöhemmin valitsee se ladattu VDI tiedosto niin se asentaa Mint:in heti päälle.

Käynnistin Linux Mint 20.

Linux Mint lähti mainiosti käyntiin ja kirjautuminen onnistui ongelmitta.

Kokeilin Linux Mint 20.

Linux Mint toimii tosi hyvin, sain siihen valmiiksi ladattua Firefox selaimen ja esim. nettiselailu toimii hyvin, kuvassa Laurean sivustot auki. Command toimii niinkuin pitääkin. Heti huomaa, että Mint on mahdollisimman lähellä Windowsia ulkonäöltään ja eroja Windowsista näkee aluksi vähemmän.

Tehtävä B)

Asensin Linux Bodhi 5.0.

Viides versio Linux Bodhi 64-bit tuli asenettua ja sen koko oli (3,05GB).

Käynnistin Linux Bodhi.

Linux Bodhi on myös Ubuntu pohjainen, eli toimii tosi hyvin ja se käynnistyi huomattavasti nopeammin kuin Mint.

Kokeilin Linux Bodhi.

Heti alkuun huomasin kuinka Linux Bodhi on nopea, kaikki ohjelmat ja jopa sama selain toimii varmaan kaksi kertaa nopeammin kuin Mint. Tässä versiossa tulee huomattavasti vähemmän resursseja käytettyä.

2.2 Tutustuminen Graafiseen käyttöliittymään

Tehtävä suoritettu, 31.03.2021
Testaaminen.

VirtualBoxilla tuli hyvin testattua sekä, Mint, että Bodhi ja asetuksien säätäminen onnistuu ihan näppärästi, sillä muistuttavat hyvin Windowsia ja MacOSia.

Tehtävä 2.3 Vaihtoehtoisia asennustapoja

Tehtävä suoritettu, 31.03.2021
A. Linux Azure-pilvessä

Azure aukesi ihan hyvin, mutta käynnistys kesti useita minuutteja. Yllättävän hidas järjestelmä.

A. Linux Azure-pilvessä

Azure komento koodin kanssa oli sellaista tappelua, ettei tee mieli ollenkaan työskennellä azuren virtual machinen kanssa. Koneeni ei tykkää etäyhteydestä, eikä reagoinut sen enempää kun lähti etäyhteys käyntiin. Turhaan tuli tuhlattua useita tunteja komentopohjan korjaamisen yhteydessä. En suosittele.

B. Linux USB-tikulle

Latasin USB tikulle RUFUS ohjelman kautta Ubuntu pohjainen Puppy-Linuxin. Koska se on pienin tiedostokooltaan, myös uskalsin nyt uhrata tikultani tärkeätä tietoa.

B. Linux USB-tikulle

Asennus vaiheessa odottelin tikun ylikirjoittamista.

B. Linux USB-tikulle

Asennus onnistui, nyt muistitikullani on Puppy Linux. Toimivuus on hyvin perinteinen, omalla työkoneella en aio käynnistää, sillä se on henkilökohtainen ja toimii päivittäin ihan kuin serveri 24/7. Jos olisi laitteisto mitä voisi uhrata kokeiluun, niin siitä vain lähtisin kokeilemaan, mutta linux asentaminen ei ole uusi juttu millään tavalla muutenkaan. Joten tehtävä on suoritettu.
